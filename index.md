This is the **demo and documentation site** for the shared studio theme — and
the visual regression surface for the studio's screenshot checks. Every page
here is built from the same layouts, includes, and components a project site
inherits.

{% include callout.html label="Adopt in ≤5 lines" body="A bare Jekyll site adopts this theme with a pinned <code>remote_theme</code>, a <code>studio:</code> config block, and nothing else. See the Design guide." %}

## The rules in one line

Clean developer-docs aesthetic; Swiss typography; near-monochrome ground.
**Color is wayfinding only** — green means *action*, blue means *information*,
orange means *alert*, and red/amber/green mean *status*. Never decoration.

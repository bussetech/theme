{%- comment -%} Studio logotype — text-only, rendered entirely from config so the theme ships with zero hardcoded org name. Reads `site.studio.logotype` (pattern " | software studio"); splits on the pipe to weight the org name bold and the descriptor regular. Params (via include): size — "lg" | "sm" (optional, default base) link — url to wrap the mark in (optional) tag — wrapping element when not a link (default "span") {%- endcomment -%} {%- assign _lt = site.studio.logotype -%} {%- if _lt == nil or _lt == "" -%}{%- assign _lt = "studio | software studio" -%}{%- endif -%} {%- assign _parts = _lt | split: "|" -%} {%- assign _org = _parts[0] | strip -%} {%- assign _desc = _parts[1] | strip -%} {%- assign _cls = "logotype" -%} {%- if include.size == "lg" -%}{%- assign _cls = "logotype logotype--lg" -%}{%- endif -%} {%- if include.size == "sm" -%}{%- assign _cls = "logotype logotype--sm" -%}{%- endif -%} {%- capture _inner -%}{{ _org }}|{{ _desc }}{%- endcapture -%} {%- if include.link -%} {{ _inner }} {%- else -%} <{{ include.tag | default: "span" }} class="{{ _cls }}" aria-label="{{ _lt }}">{{ _inner }} {%- endif -%}

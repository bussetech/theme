{%- comment -%} Sortable table. Params: columns — array of column objects: key (field name in each row) label (header text) numeric (optional: right-align + mono + numeric sort) title (optional: header title attribute, e.g. to expand an abbreviated label) rows — array of row objects. Cell display value = row[key]. Optional row fields: _sort — sort value when display is formatted (e.g. tokens "71k" displays, 71382 sorts) _url — links the FIRST column's cell default_sort — key pre-sorted at render (progressive enhancement: the table arrives sorted; JS only re-sorts) default_desc — true to pre-sort descending (default true) caption — accessible table caption (visually hidden) compact — true for rail-width contexts (xs type, tight cells) Sorting needs `sortable_tables: true` in the page front matter to emit assets/js/sortable-table.js (the map-scripts pattern).
